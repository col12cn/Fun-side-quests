Paper Portfolio Tracker - AI Strategy Engine

This upgraded file keeps your paper portfolio tracker but replaces the fixed rebalance template with a decision engine that scores market signals and portfolio constraints before logging any paper trade. The starting portfolio and all notional values are now denominated in USD to match the tickers. This prototype is currently simplified to fetch quotes only so the portfolio tab works reliably.

What changed

The old prototype used a fixed rule-based rebalance. This version adds a portfolio snapshot, Finnhub-powered live quotes and candle history, ticker news fetching, Perplexity-backed structured trade proposals, deterministic risk checks, and paper execution with separate AI, risk, and trade logs.

Migration-safe This USD version auto-converts older EUR/local state fields when possible so existing saved portfolios continue to refresh.
Preferred Set a backend endpoint that calls Perplexity securely from your server.

Gemini news fetch

This tab uses Gemini with Google Search grounding to fetch recent ticker news separately from quote refresh. JSON mode is disabled here because Google Search grounding cannot be combined with an application/json response mime type on this endpoint.

How it works

1. Paste your Google AI Studio key. 2. Click Fetch news with Gemini. 3. The app asks Gemini for recent news on all portfolio tickers. 4. Gemini returns JSON with headline, summary, sentiment, and source URL. 5. The results are cached in your browser and shown below.
